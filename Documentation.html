Introduction

Modern Chat UI Kit is a comprehensive Flutter UI toolkit designed to create stunning messaging applications. Built with Flutter's latest features, it provides a complete set of pre-built screens and components that follow Material Design principles while maintaining a modern, customizable aesthetic.

The kit includes 11 meticulously crafted screens including chat conversations, user profiles, settings, and call interfaces. Each component - from customizable chat bubbles and user avatars to typing indicators and story rings - is designed for seamless integration. The UI Kit features smooth animations, responsive layouts, and a consistent design language across all screens.

Key features include: • Complete chat interface with message status indicators • Story/Status viewing system • Audio/Video call screens • User authentication flows • Profile management • Customizable themes and styles • Responsive design for all screen sizes

Project Structure

The Modern Chat UI Kit follows a clean, modular architecture designed for scalability and maintainability. Here's a detailed breakdown of the project structure:

📁 Core Files

Entry point and app configuration
  • lib/main.dart - Application entry point
  • lib/modern_chat/app.dart - App configuration and theme setup

🎨 Theme

App-wide styling and theme configuration
  • theme/app_colors.dart - Global color constants
  • theme/app_text_styles.dart - Typography and text styles

🧩 Widgets

Reusable UI components used across screens
  • widgets/animated_list_item.dart - Animated list item transitions
  • widgets/chat_bubble.dart - Chat message bubbles with reactions
  • widgets/message_input.dart - Chat input with attachments
  • widgets/search_bar.dart - Animated search functionality
  • widgets/shimmer_loading.dart - Loading state animations
  • widgets/story_ring.dart - Story/Status ring indicators
  • widgets/user_avatar.dart - User avatars with online status

📊 Models

Data models with sample data for demonstration
  • models/call.dart - Call model with audio/video call data
  • models/chat_message.dart - Message model for chat functionality
  • models/chat_user.dart - User model with profile information
  • models/story.dart - Story/Status model with sample stories

📱 Screens

Feature-specific screens and UI views
  • screens/splash_screen.dart - App launch and initialization
  • screens/login_screen.dart - User authentication
  • screens/signup_screen.dart - New user registration
  • screens/home_screen.dart - Main chat list with stories and calls
  • screens/chat_detail_screen.dart - Individual chat conversations
  • screens/call_screen.dart - Audio and video call interface
  • screens/profile_screen.dart - User profile management
  • screens/settings_screen.dart - App preferences and settings
  • screens/new_chat_screen.dart - Create new conversations
  • screens/shared_content_screen.dart - View shared media and files
  • screens/status_view_screen.dart - View and interact with stories

🛠️ Utils

Helper functions and utilities
  • utils/page_routes.dart - Custom page transition animations (Fade, Slide, Scale)
Best Practices:
  • Each feature follows a modular architecture for better maintainability
  • Clear separation of concerns between UI, logic, and data layers
  • Consistent naming conventions throughout the project
  • All files are organized under the modern_chat/ directory for better encapsulation
  • Sample data is included in model files for demonstration purposes
  • Custom transitions and animations for enhanced user experience
