Introduction

Modern Chat UI Kit is a sophisticated and contemporary set of Flutter UI components designed specifically for chat and messaging applications. This comprehensive kit includes complete chat templates, ready-to-use messaging widgets, and stunning screens that follow modern design principles.

Built with Flutter's latest features, Modern Chat UI Kit offers a perfect blend of aesthetics and functionality. The kit includes essential messaging features like chat bubbles, user avatars, typing indicators, and message status indicators, all crafted with attention to detail and modern design trends.

Prerequisites

Add this to your pubspec.yaml:

dependencies: modern_chat_ui: ^1.0.0

Or run this command:

flutter pub add modern_chat_ui
  • Flutter SDK: 3.0 or higher
  • Dart SDK: 2.17 or higher
  • Android Studio / VS Code with Flutter extension
Note: For support and inquiries, contact us at: support@flutterfusion.dev

Step 1: Get the Flutter SDK

Download and extract the Flutter SDK:

Download Flutter SDK and extract to C:\src\flutter

Do not install Flutter in a directory like C:\Program Files\ that requires elevated privileges.

Step 2: Update your PATH

  1. From the Start search bar, enter 'env' and select "Edit environment variables for your account"
  2. Under User variables check if there is an entry called Path:
    • If exists: Append the full path to flutter\bin using ; as a separator
    • If doesn't exist: Create a new user variable named Path with the full path to flutter\bin
Note: Close and reopen any existing console windows for these changes to take effect.

Step 3: Verify Installation

Run flutter doctor to verify your installation and see any remaining dependencies you may need:

flutter doctor

This command checks your environment and displays a report of the status of your Flutter installation.

  1. Download SDK and extract the file
  2. Move the extracted flutter folder to your desired location (e.g., Documents/flutter)

Step 2: Update your path

You have two options to update your PATH:

Option 1: For current terminal session
export PATH="$PATH:`pwd`/flutter/bin"
Option 2: Update permanently
  1. Create/open .bash_profile:
    sudo open -e $HOME/.bash_profile
  2. Add the following line:
    export PATH="$PATH:[PATH_TO_FLUTTER_GIT_DIRECTORY]/flutter/bin"
  3. Refresh PATH variables:
    source $HOME/.bash_profile

Getting Started (Build & Run)

Important: All below steps must be followed to build and run application

Download Project

Download and find your project folder, use your preferred IDE (Android Studio / Visual Studio Code / IntelliJ IDEA) to run the project.

Get Dependencies

After you loaded project successfully, run the following command in the terminal to install all the dependencies listed in the pubspec.yaml file in the project's root directory or just click on Pub get in pubspec.yaml file if you don't want to use command.

flutter pub get

Build and Run App

  1. Locate the main Android Studio toolbar.
  2. In the target selector, select an Android device for running the app.
  3. Click the run icon in the toolbar, or invoke the menu item Run > Run.
Note: If you don't use Android Studio or IntelliJ you can use the command line to run your application using:
flutter run

Try Hot Reload

Flutter offers a fast development cycle with Stateful Hot Reload, the ability to reload the code of a live running app without restarting or losing app state. Make a change to app source, tell your IDE or command-line tool that you want to hot reload, and see the change in your simulator, emulator, or device.
